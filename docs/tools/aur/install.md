With `yay` (replace by your favorite AUR helper):

```bash
yay -S bloop
```

If you want to use the [Systemd service](usage.md), also install `bloop-systemd`:

```bash
yay -S bloop-systemd
```

### Requirements

1. **Java 8 or higher**
1. **Python 2 or 3** (installation script and CLI client are written in Python)
